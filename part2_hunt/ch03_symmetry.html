Chapter 3 · Part II Symmetry & the Order-7 Door

If Conway's graph has any symmetry, assuming it shrinks the search to a tiny orbit matrix. Order by order, the symmetric cases are being closed — $11$ gone, $3$ gone, the most symmetric Cayley case gone. One door stands ajar: an automorphism of order $7$, in a single rigid configuration that no desktop can exhaust.

3.1   The orbit-matrix engine

Assume $\Gamma=\mathrm{SRG}(99,14,1,2)$ has an automorphism $\sigma$ of prime order $p$. It splits the $99$ vertices into $f$ fixed points and $\tfrac{99-f}{p}$ orbits of size $p$, and the adjacency matrix collapses to a small orbit matrix obeying the Behbahani–Lam column equations. Enumerate those integer matrices exhaustively: none means no graph has that symmetry; one that lifts is Conway's graph. Because $99 = 3^2\cdot 11$, the orders that divide $99$ are $3,9,11,33,99$; other primes can still act, but only with fixed points.

Theorem (orders 11 and 3, and no abelian Cayley)

No $\mathrm{SRG}(99,14,1,2)$ admits a semiregular automorphism of order $11$ (all three feasible configurations UNSAT), nor an automorphism of order $3$. Moreover it is not a Cayley graph on either abelian group of order $99$ ($\mathbb{Z}_{99}$ and $\mathbb{Z}_3\times\mathbb{Z}_{33}$): both prescribed-difference-set encodings are UNSAT. $\square$

3.3   The order-7 door

Seven does not divide $99$, but an order-$7$ automorphism can still act — with fixed points. Cesarz and Woldar (2023) proved there is exactly one possibility: a single fixed point and $14$ orbits of size $7$, the configuration $[1, 7^{14}]$. They could not decide whether such a graph exists; neither, yet, can we.

The unique order-7 configuration $[1,7^{14}]$ — open

Enumerating the $[1,7^{14}]$ orbit matrices, sharpened with the Cesarz–Woldar $\Gamma_2$ orbit-valency constraints, the dedicated solver ran 281 million nodes in one hour without exhausting the case — stack depth still climbing. The order-$7$ door is neither a proof of non-existence nor a graph: it is cluster-scale. Closing it UNSAT would strengthen Cesarz–Woldar; lifting one matrix would win Conway's prize.

3.4   What remains

The involutions are the largest unfinished block: an order-$2$ automorphism fixes an odd number of points, giving $46$ feasible $(f,o)$ configurations, each a separate orbit-matrix run. Orders $9$, $33$, $99$ (cyclic, non-prime) and the small non-abelian groups are still to be swept. The pattern so far, though, is unmistakable and matches Conway's own intuition:

Standing structure

Every automorphism order closed so far returns nothing. If Conway's graph exists, its symmetry is small — most likely an involution, the lone order-$7$ action, or nothing at all. That pushes the decisive work toward the asymmetric methods of Chapters 4 and 5.
